Offshore Wind Energy Forecasting

A machine learning pipeline for predicting offshore wind power output using KNMI weather data and SCADA measurements from Dutch wind farms.

View Forecast Dashboard →
🌊

Dutch Offshore Wind Farms

Forecasts target the Hollandse Kust Zuid (HKZ) and Hollandse Kust Noord (HKN) wind farm zones in the North Sea.

📡

Data Sources

Pipeline ingests KNMI meteorological data, ENTSOE energy production records, and SCADA turbine measurements for model training.

🤖

ML Predictions

Hourly power output predictions are generated by a trained model and served via a REST API to this dashboard.
